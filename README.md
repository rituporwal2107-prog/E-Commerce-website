# 🛒 E-Commerce Website

Welcome to my E-Commerce Website project! This is a modern web application designed for online shopping.

---

## 🚀 Features (Khasiyat)
* *User Authentication:* Simple login and registration pages.
* *Product Catalog:* Beautiful display of items with images and details.
* *Shopping Cart:* Easy to add and manage items before checkout.

## 🛠️ Technologies Used
* *Frontend:* HTML5, CSS3, JavaScript
* *Images & Assets:* Categorized product images for realistic UI.

---

## 📁 Project Structure
* e login - Login & Sign-up pages
* Ecommerce / Shopping - Core store functionalities
* product detalis - Dedicated view for single items
* shopping card - Cart management logic
